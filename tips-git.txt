
iniciar Git: Crea staging area y repositorio local
==================================================
git init

Agrega archivos al staging area
===============================
git add .
git add archivo

Traslada archivos desde staging area al repositorio local
=========================================================
git commit -m "nombre del commit"

Usar rama main
================
git branch -M main


git remote add origin https://github.com/aconlledo/proximahora.git #Esto para la primera vez que se hace un commit
git push -u origin main

git remote add origin https://github.com/aconlledo/proximahoradis.git #Esto para la primera vez que se hace un commit
git pull -u origin main

Archivos considerados en seguimiento antes del commit
=====================================================
git status -s

git log --oneline  Muestra los cambios en el repositorio (despues de commint)
git reset --hard <codigo-instantanea>  Restaura archivo (contrario a commit)
